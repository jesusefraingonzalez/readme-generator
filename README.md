  ## Installation
  ---
  - Fork the repository from github
  - Run the following commands in the cli
  ```
  npm install
  node index.js
  ```
  - Answer the prompts
  <br><br>
  ## Usage
  ---
  Use this generator to generate a professional looking README for your projects.
